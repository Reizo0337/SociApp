# SociApp
Este proyecto consiste en una plataforma integral de gestión para una Asociación de Vecinos, diseñada para digitalizar y centralizar la administración de socios, actividades, proyectos y la contabilidad de la entidad.

## Estructura del Proyecto

El repositorio se divide en dos componentes principales:

- **Backend**: API y lógica del servidor (construido con NestJS).
- **Frontend**: Interfaz de usuario para la gestión de la plataforma.

## Instalación y Configuración

### Requisitos Previos

- Node.js
- npm o yarn

### Backend

1. Navega al directorio del backend:
   ```bash
   cd backend
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run start:dev
   ```

### Frontend

1. Navega al directorio del frontend:
   ```bash
   cd frontend
   ```
2. Instala las dependencias e inicia la aplicación:
   ```bash
   npm install
   npm start
   ```
